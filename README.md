# 🎧 Moodify — Music That Understands You

> *I wanted to explore how face detection APIs actually work in real-world apps…*
> *So instead of building something boring, I built something personal.*

---

## 💡 The Idea

I’ve always noticed how music changes my mood — but what if an app could **understand my mood automatically and suggest the right songs instantly?**

That curiosity led me to build **Moodify**.

A project where I combined:

* Face Detection APIs
* MERN Stack
* Real-time mood analysis

To create something simple but powerful:
👉 *Music that adapts to how you feel.*

---

## 🚀 What Moodify Does

Moodify detects your facial expression using your camera and instantly recommends songs based on your mood.

No typing. No searching.
Just click → detect → listen.

---

## ⚙️ Core Features

* 🎭 **Real-time Mood Detection**
  Detects facial expressions using Google Face Detection API

* 🎵 **Mood-Based Song Recommendations**
  Suggests songs that match your detected emotion

* 📸 **Camera Integration**
  One-click detection with live camera access

* ⚡ **Fast & Simple UX**
  Minimal steps from detection to music

---

## 🧠 How It Works

1. Open the app
2. Click on **Detect Mood**
3. Allow camera access
4. Face is analyzed using Google API
5. Mood is detected (happy, sad, etc.)
6. Songs are recommended instantly
